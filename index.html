Distillery

This is a bare-bones program analysis library that exposes simple interfaces to control flow graphs and concrete analyses.

Building

Build by making a build directory (i.e. build/), run cmake in that dir, and then use make to build the desired target.

Example:

1 > mkdir build && cd build
2 > cmake .. -DCMAKE_BUILD_TYPE=[Debug | Coverage | Release]
3 > make
4 > ./main
5 > make test # Makes and runs the tests.
6 > make coverage # Generate a coverage report.
7 > make doc # Generate html documentation.
